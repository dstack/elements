Use a button to represent a user action.

<!-- The default button style --></p> <button class="pe-btn">Default</button> <!-- The primary action for a given context, such as the default action in a dialog --> <p><button class="pe-btn pe-btn--primary">Primary</button>

Disabled

Buttons can be disabled when an action is not available to the user for the current context.

</p> <p><button class="pe-btn" disabled>Default</button></p> <p><button class="pe-btn pe-btn--primary" disabled>Primary</button>

For non-button elements, use pe-btn--disabled.

</p> <p><div class="pe-btn pe-btn--disabled">Not a &lt;button&gt;</div>

Sizes

Buttons can be made smaller or larger.

</p> <p><button class="pe-btn pe-btn--small">Small</button></p> <p><button class="pe-btn pe-btn--large">Large</button>

Links

In some cases, it may be necessary to style a button as a link.

</p> <p><button class="pe-btn pe-btn--link">Link</button>
